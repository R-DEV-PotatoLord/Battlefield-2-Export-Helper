# Battlefield-2-Export-Helper
Tools to be used in conjunction in BF2 Blender Tools

Installation instructions: 
Install "bf2exporthelper.py" from disk.
Alternatively convert that file only to .zip and drag it into an opened Blender window.

Functionality in v0.2X includes generating hierarchy required for .bundledmesh export and populating said hierarchy with automatically generated LODs. Support for .staticmesh is planned but not yet implemented.
